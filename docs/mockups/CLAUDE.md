## Project Overview

This is the IgniteHealth app repository - a comprehensive health tracking platform that helps users discover correlations between lifestyle choices and health outcomes. The app focuses on visual documentation of food consumption combined with detailed health check-ins to identify patterns that impact wellbeing.

## Core Architecture

### Project Structure
The repository contains primarily documentation and HTML prototypes for a health tracking application:

- **Documentation Framework**: Comprehensive markdown files defining the complete app specification
- **Interactive Prototypes**: HTML mockups demonstrating UI/UX concepts and user flows
- **Health Tracking System**: Detailed frameworks for morning/evening check-ins across 7 health domains

### Key Framework Components

1. **Health Tracking Sections** (7 core domains):
   - Vital Rhythms (Sleep & Energy)
   - Body Signals (Physical Symptoms) 
   - Fuel & Flow (Nutrition & Digestion)
   - Movement Matrix (Exercise & Recovery)
   - Mood & Mindset (Mental & Emotional Wellbeing)
   - Medicine Management (Medications & Treatments)
   - Menstrual Mapping (Hormonal Patterns)

2. **Photo Documentation System**:
   - Visual food/supplement tracking with AI recognition
   - Three-tier confidence system (High/Good/Estimation modes)
   - Integration with USDA, Open Food Facts, and Nutritionix databases

3. **Correlation Engine**:
   - Symptom clustering across health domains
   - Statistical correlation detection with confidence scoring
   - Pattern recognition for lifestyle-health relationships
