# Epic 5 — Lab Results Ingestion & Analysis

## Goal
Ingest lab reports (PDF/images), extract structured observations, validate with human-in-the-loop, and store analytes for longitudinal tracking.

## Included (v1 scope)
- Upload lab report (PDF/image)
- Extraction pipeline (OCR/LLM) + staging
- Human review & publish
- Chart analytes over time (via Epic 6)

## Excluded
- Direct lab integrations (future)
- Medical advice/diagnosis (never)
