# Epic 4 — Photo-Based Health Tracking

## Goal
Enable users to capture and review visual changes over time (face/body/targeted photos), linked to time-anchored entries.

## Included (v1 scope)
- Photo capture flows (standalone journal + attach-to-entry)
- Metadata and secure storage
- Time-series comparison views (basic)

## Excluded
- Automated diagnosis
- Facial recognition
