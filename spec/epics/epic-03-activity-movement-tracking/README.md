# Epic 3 — Activity & Movement Tracking

## Goal
Track physical activity and movement (intentional or incidental) as time-anchored events to correlate with symptoms and recovery.

## Included (v1 scope)
- One-screen activity log
- Duration + perceived intensity
- Optional activity type/category
- Optional notes

## Excluded
- Full workout planning (future)
- Wearable integrations (future)

## Notes
Avoid blending “how I felt” (Epic 1) with “what I did” (this epic).
