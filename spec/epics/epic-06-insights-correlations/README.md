# Epic 6 — Insights & Correlations

## Goal
Help users see patterns across journals, intake, activity, photos, and labs.

## Included (v1 scope)
- Trend charts and comparisons
- Filters by date range and data type
- Correlation-friendly views (side-by-side charts)

## Optional / Pro (planned)
- AI-assisted summaries and “help me understand” explanations
- Non-diagnostic insights only (pattern detection, summarization)

## Excluded
- Diagnostic claims
- Treatment recommendations
