# Epic 8 — Privacy, Security & Trust

## Goal
Provide secure access and privacy guarantees appropriate for sensitive health data (HIPAA-ready posture).

## Included (v1 scope)
- Registration/login (passkeys preferred)
- Account recovery
- PII isolation and access controls
- Auditability (at least for sensitive operations)
- Data export and deletion requests (baseline)

## Excluded
- Formal HIPAA certification (v1 posture is readiness)
